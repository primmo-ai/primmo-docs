# Installation

Cette section décrit comment créer un compte utilisateur, installer le complément Outlook, installer le complément Word, autoriser le microphone et configurer la synchronisation Outlook.

---

## Création de compte utilisateur

La création de compte s’effectue depuis la **page d’inscription** (signup) de l’application Primmo, accessible depuis le site ou depuis le complément une fois celui-ci ouvert.

1. Renseignez votre **adresse e-mail** et **confirmez-la**.
2. Choisissez un **mot de passe** (au moins 6 caractères).
3. Cliquez sur **Créer un compte**.

**Important : notez votre mot de passe.** Conservez-le dans un endroit sûr ; il vous servira pour vous connecter à l’application web et aux compléments Outlook et Word. Si vous l’oubliez, utilisez la fonction « Mot de passe oublié » sur la page de connexion.

## Installation Outlook

Le complément Primmo s’installe **depuis Outlook**, en ajoutant un complément à votre messagerie.

### Vidéo explicative

*(Emplacement prévu pour une vidéo explicative : installation du complément Outlook.)*

### Depuis Outlook : ajout de compléments

1. Ouvrez **Outlook** (application de bureau ou Outlook sur le web).
2. Allez dans **Obtenir des compléments** (ou **Add-ins** / **Store** selon votre version).
3. Recherchez **Primmo** (ou « Primmo - IA Notariale »).
4. Sélectionnez le complément Primmo et validez l’**installation**.
5. Le complément apparaît dans la barre latérale (ou dans le ruban, selon la configuration). Vous pouvez l’**épingler** pour le garder toujours accessible.

### Si Primmo n’apparaît pas : réinitialiser le cache

Si le complément ne s’affiche pas ou ne se charge pas correctement après installation :

1. **Fermez complètement Outlook** (y compris les processus en arrière-plan si nécessaire).
2. **Videz le cache** des compléments Office (selon votre environnement, via les options de réparation Office ou en supprimant les données de cache du complément).
3. **Rouvrez Outlook** et réessayez d’ouvrir Primmo depuis « Mes compléments » ou le menu des compléments.

En cas de blocage persistant, contactez le support Primmo en indiquant votre version d’Outlook (bureau ou web) et le message d’erreur éventuel.

## Installation Word

Le complément Primmo pour Word s’installe via un **programme d’installation** (setup) fourni par Primmo.

### Vidéo explicative

### Lien de téléchargement du setup

Le fichier d’installation (setup exe) du complément Word est fourni par Primmo (lien de téléchargement communiqué par votre étude ou disponible dans votre espace compte). Téléchargez le fichier sur votre poste avant de lancer l’installation.

### Étapes d’installation

1. **Fermez Word** complètement (toutes les fenêtres Word et processus associés).
2. Lancez le **fichier setup** (exe) téléchargé.
3. Suivez les instructions à l’écran (acceptation des conditions, choix du répertoire si proposé).
4. À la fin de l’installation, redémarrez le poste si l’assistant le recommande, puis ouvrez Word : le complément Primmo doit apparaître (ruban ou menu selon la version).

## Installation micro (autorisation microphone)

L’assistant Primmo peut utiliser le **microphone** pour la saisie vocale et la transcription (demandes au chat par la voix). L’autorisation est demandée **lors de la première utilisation** dans Outlook et dans Word.

### À l’installation dans Outlook et dans Word

Lorsque vous utilisez pour la première fois la fonction **micro** dans le chat :

1. Cliquez sur l’icône **microphone** dans la zone de saisie du chat.
2. Une invite du navigateur ou de l’application vous demande l’**accès au microphone**.
3. **Cliquez sur « Autoriser »** pour que Primmo puisse enregistrer et transcrire votre voix.

Sans cette autorisation, la saisie vocale ne fonctionnera pas. Vous pourrez toujours utiliser la saisie au clavier. Sur Outlook Web, si l’invite n’apparaît pas, actualisez la page et réessayez ; vérifiez aussi les paramètres du navigateur (autorisations pour le site Primmo).

## Synchronisation Outlook

Une fois le complément Outlook installé et votre compte créé, la **synchronisation** avec Primmo permet de :

- voir et attacher l’e-mail en cours au contexte du chat ;
- enregistrer des e-mails dans Primmo (dossiers) ;
- garder un lien cohérent entre votre messagerie et vos dossiers Primmo.

Assurez-vous d’être **connecté** avec le même compte Primmo dans le complément. La connexion se fait automatiquement après ouverture du complément si vous avez déjà une session active, ou via la page de connexion du complément. Une fois connecté, les données nécessaires (dossiers, e-mails enregistrés) sont synchronisées selon les actions que vous effectuez dans l’interface.
