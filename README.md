# Primmo Documentation

This repository holds the documentation for Primmo, published with [GitBook](https://www.gitbook.com/).

## Structure

- **README.md** – This file; landing page for the space.
- **SUMMARY.md** – Table of contents used by GitBook for navigation.
- **/** – Other Markdown files and folders as referenced in `SUMMARY.md`.

## Editing

- Edit content in this repo and push to your default branch; GitBook (when GitHub Sync is enabled) will sync from the chosen branch.
- Or edit in GitBook’s UI; merged change requests are committed back to this repo.

## Local setup

```bash
git clone <your-repo-url> primmo-docs
cd primmo-docs
```

No build step is required; GitBook hosts and renders the content.
