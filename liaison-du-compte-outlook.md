# Liaison du compte Outlook

## Installation

{% hint style="info" %}
Consultez [#installation-outlook](installation.md#installation-outlook "mention") pour comprendre comment activer la liaison de votre compte Outlook.
{% endhint %}

## Consultation du calendrier

Une fois la liaison Outlook activée, Primmo est en mesure de consulter vos disponibilités de calendrier.

{% hint style="info" %}
La consultation du calendrier LRA est uniquement possible si vous avez activé la synchronisation entre votre calendrier LRA et Outlook.
{% endhint %}

## Classement automatique de vos emails

Une fois la liaison Outlook activée, Primmo essaie de classer automatiquement vos emails et pièces jointes dans le bon dossier.

{% hint style="info" %}
Les emails et pièces jointes ne sont pas rangés dans votre dossier LRA par défaut.
{% endhint %}
