# Personnaliser votre assistant

## Le menu personnalisation

Depuis le menu Paramètres, accédez au champ "Personnalisation".

Ici, vous pouvez indiquer vos préférences afin qu'elles soient respectées par votre assistant dans le cadre des réponses à vos demandes.

## La mémoire

Dans le cadre d'une conversation, vous pouvez demander à l'assistant de se souvenir d'informations importantes. L'assistant mettre à jour sa mémoire pour tâcher de respecter ces indications lors des prochaines demandes.
